Fais-moi une page web en javascript et HTML pour créer un chronomètre de Rubik's Cube a utiliser sur téléphone en mode portrait. Quand on entre sur la page web il y a une description (qui prends la taille de tout l'écran) : " Chronomètre pour Rubik's Cube

au-revoir

Cliquez sur le bouton commencer> et appuyer avec les deux doigts pour commencer le chronomètre dès que le premier doigt est enlevé le pro le chronomètre commence et dès qu'un doigt se repose sur l'écran le chronomètre s'arrête. " Dès que la personne clique sur la description la description s'efface et la page principale s'affiche : Il y a le titre "Rubikstimer" en haut de l'écran. Le fond d'écran est bleu marine. Il y a un bouton rouge en haut à gauche de l'écran avec écrit "comencer" dessus le joueur doit cliquer dessus et mettre ses deux doigts sur l'écran. Quand le joueur a cliqué sur le bouton, le bouton se met en gris. Dès que le premier doigt est enlevé de l'écran un chronomètre commence et est visible au milieu de l'écran, le chronomètre s'arrête et le bouton redevient rouge quand le joueur repose un doigt sur l'écran. Pour recommencer un autre chronomètre le joueur doit recliquer sur le bouton (si il est gris ça ne marche pas) et faire la même démarche que la première fois. Il il y a un autre bouton avec écrit "tableau des scores" dessus quand le joueur clique dessus et que le bouton commencer est rouge tous les scores s'affichent sur une page blanche qui prend tout l'écran avec le numérotation (1,2,3...) et le score à sa droite ( par ex : 1- 53s ). Il y à tous les scores en colonne. Il y a une croix à droite de chaque score qui sert à supprimer le score à sa gauche si le joueur clique dessus. Il y a une flèche en haut à gauche de la page pour retourner à la page principale quand le joueur clique dessus.
